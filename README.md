# HỆ THỐNG QUẢN LÝ CÂY XANH

## Giới thiệu

Đây là đồ án xây dựng website quản lý cây xanh trong Trường Đại học Trà Vinh.

## Công nghệ sử dụng

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap 5
- Leaflet
- Chart.js

## Chức năng

- Đăng nhập và phân quyền
- Quản lý cây xanh
- Quản lý khu vực
- Quản lý lịch sử chăm sóc
- Tra cứu cây xanh
- Phản ánh sự cố
- Bản đồ cây xanh
- Thống kê
- Cảnh báo
- Xuất dữ liệu Excel
