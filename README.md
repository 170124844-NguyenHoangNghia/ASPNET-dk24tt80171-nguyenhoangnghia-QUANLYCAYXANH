# Hệ thống quản lý cây xanh

## Giới thiệu

Đây là đồ án xây dựng hệ thống website quản lý cây xanh trong Trường Đại học Trà Vinh.

## Công nghệ sử dụng

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap
- Chart.js
- Leaflet
- OpenStreetMap

## Chức năng

- Đăng nhập và phân quyền
- Quản lý cây xanh
- Quản lý khu vực
- Quản lý lịch sử chăm sóc
- Tra cứu cây xanh
- Phản ánh sự cố
- Bản đồ cây xanh
- Thống kê
- Cảnh báo
- Xuất dữ liệu Excel
